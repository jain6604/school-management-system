# 🎓 School Management System

A full-stack School Management System built using **React.js** (Frontend) and **ASP.NET Core Web API** (Backend).

---

## 🚀 Features

- 📄 Student Admission System
- 📋 Pending Admissions Approval / Rejection
- 👨‍🎓 Student Information System (SIS)
- 📊 Marks & Result Management
- 👨‍🏫 Staff Management
- 📈 Analytics Dashboard
- 👨‍👩‍👧 Parent Portal
- 📢 Result Publishing System

---

## 🛠 Tech Stack

### Frontend
- React.js
- JavaScript
- HTML/CSS

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQLite / SQL Server
